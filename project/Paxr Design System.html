01 — Colour

A palette of resolution,
not decoration.

Deep is the voice. Stone is the room. Jade is reserved for one thing: the moment something is handled. Never decorative, never more than one jade element per screen.

Paxr Deep
#0E1B29 · rgb(14 27 41)
Primary dark. All dark UI. Text on light. The voice of the product.
Paxr Jade
#3EC99A · rgb(62 201 154)
Resolution only. CTAs, confirmations, active status. One element per screen — maximum.
Paxr Stone
#F6F5F1
App background. Warm, never pure white.
Paxr Pale
#EAE9E4
Dividers, subtle borders, card surfaces on light.
Paxr Depth
#1C2E40
Card surfaces on dark UI. One level lighter than Deep.
Paxr Mist
#8FA3B1
Secondary text, labels, metadata. Never primary.
Paxr Amber
#E8A020
Expiry warnings only. Never decorative.
Paxr Error
#C0392B
Expired, errors, destructive actions only.
Jade is sacredOne jade element per screen, maximum. Used on resolution, not anywhere a link might be hovered.
Never pure whiteStone is warm. A stark #FFF next to it reads as a bug, not a surface. Cards lift through hairlines instead.
No gradients, everDepth is built from adjacent flats — Deep beside Depth, Stone beside Pale. Colour fades are a tell.

05 — Components

The small pieces that
carry the whole thing.

Every component exists for a specific moment in the warranty lifecycle. Nothing is decorative. Hover the primary buttons to see motion — every transition uses the same easing curve.

Buttons

Three flavours. Jade is always the resolution action — "Scan", "Renew", "Done". Deep is secondary. Ghost is tertiary, used for less-urgent options.

07 — Voice

Write like a competent friend,
never a helpful robot.

Paxr's voice is short, steady, and already handled. No exclamation marks, no emojis in the app, no passive voice, no apology. The agent doesn't get excited — it gets things done.

✓ Write like this
1
Confirmation
Done. Covered until Nov 2027.
Past tense. Period, not exclamation.
2
Warning
Expires in 23 days. Renew?
Fact, then question. No pressure.
3
Empty state
No warranties yet. Scan a product to start.
4
Error
QR code not recognised. Add manually instead.
Name the problem, offer the exit.
5
CTA
Scan QR code · Renew now · View warranty
Verb + object. No "Click here".
✗ Never write like this
1
No exclamations
Warranty registered!
The product is calm. Punctuation should be too.
2
No passive voice
Your warranty has been successfully registered.
→ "Registered."
3
No jargon
Initiate warranty registration process.
4
No emojis in product UI
Hi there! 👋 Just a quick reminder…
Emojis undo the whole premise.
5
No filler
We just wanted to let you know that…
Say the thing. Leave.
